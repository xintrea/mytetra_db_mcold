1. ставим go


sudo apt-get install golang


2. настраиваем GOPATH


vim /etc/profile


export GOPATH=/home/mcold/workspace/go


:source /etc/profile


3. настраиваем PATH для запуска бинов


export PATH=$PATH:$GOPATH/bin


4. пишем программу в формате


$GOPATH/src/<имя программы>/<файл>.go


5. собираем бинарник


go install <имя программы>


6. запускаем


<имя программы>
