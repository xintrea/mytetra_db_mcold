Выполнение SQL скрипта в MySQL

Posted on Апрель 16, 2012

Ответ на данный вопрос нашёлся здесь.

> mysql -h host -u user -p password < sql_filename

Пароль указывать в открытом виде не обязательно. Можно просто указать -p без строки, пароль будет запрошен:

Например

> mysql -h localhost -u root -p < sql_filename

Дополнительно, может потребоваться имя базы данных в параметрах. Тогда скрипт запускается так:

> mysql -h localhost -D dbname -u root -p < sql_filename
