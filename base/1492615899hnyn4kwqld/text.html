## bash-команды


### Справка


```sh

man ls

```




Открытие содержимого

```sh

cat ~/text.txt

```

Конвертация rpm пакета

Convert rpm package using command

```sh

sudo alien -k --scripts some-rpm-package.rpm

```

Install it using this command

```sh

sudo dpkg -i some-rpm-package.deb

```


### История введенных команд

```sh

history

```


### Процессы ps

```sh

ps axu | grep <опознавательное слово>

kill <№ процесса>

```

показать все запущенные процессы

```sh

top

```

### Поиск


искать pattern в files

```sh

grep pattern files

```

найти все файлы с именем file

```sh

locate file

```

искать рекурсивно pattern в dir

```sh

grep -r pattern dir

```


убить все процессы с именем proc *

```sh

killall proc

```


### Файлы


вывести первые 10 строк file

```sh

head file

```

вывести последние 10 строк file

```sh

tail file

```

Какая app будет запущена по умолчанию

```sh

whichapp

```


Место расположения программы

```sh

whereapp

```

форматированный список со скрытыми каталогами и файлами

```sh

ls -al

```

вывести содержимое file

```sh

more file

```

удалить каталог dir

```sh

rm -r dir

```

удалить форсированно каталог dir *

```sh

rm -rf dir

```

### Копирование


скопировать dir1 в dir2; создаст каталог dir2, если он не существует

```sh

cp -r dir1 dir2

```

### Crontab


вызов стандартного редактора для редактирования списка автоматически выполняемых задач

```sh

crontab -e

```
