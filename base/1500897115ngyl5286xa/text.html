Запуск, остановка и перезапуск сетевых интерфейсов в ОС Ubuntu Server 14.04, Debian 7


Перед началом настройки сетевых параметров вычислителей, необходимо уточнить правильность подключения его сетевых интерфейсов. Далее в процессе настройки будет использоваться следующая конфигурация:

- интерфейс eth0 вычислителей подключен к сети 192.168.108.0/24;

- интерфейс eth1 вычислителей подключен к сети 192.168.109.0/24;

Необходимо привести файл /etc/network/interfaces на основном сервере s1vm1 к следующему виду:

# The loopback network interface
auto lo
iface lo inet loopback

auto eth0

iface eth0 inet static

  address 192.168.108.2

  netmask 24

 

auto eth1

iface eth1 inet static

  address 192.168.109.2

  netmask 24



С некоторых пор команды запуска, остановки, перезапуска сетевых интерфейсов не срабатывают так, как мы того ждём.

#service networking start
#service networking stop
#service networking restart

Вместо них теперь используются новые команды:

запуск сетевого интерфейса eth0

ifup eth0


остановка сетевого интерфейса eth0

ifdown eth0


и, соответственно, перезапуск

ifdown eth0 && sudo ifup eth0
