RPAD ФУНКЦИЯ

Узнайте, как использовать Oracle / PLSQL функцию RPAD с синтаксисом и примерами.

Описание

Функция Oracle / PLSQL RPAD дополняет с правой части строки определенный набор символов (при не нулевом string1).

Синтаксис

Синтаксис функции Oracle / PLSQL RPAD:

RPAD( string1, padded_length, [ pad_string ] )

Параметры или аргументы

string1 строка, для дополнения символов (с правой стороны).

padded_length количество возвращаемых символов. Если padded_length меньше исходной строки, то функция RPAD обрежет строку с размером padded_length.

pad_string не является обязательным. Это строка, которая будет подгоняются по правой стороне string1. Если этот параметр не указан, функция RPAD дополнит пробелы к правой стороне string1.

Примечание

  • Смотрите также: LPAD функция.

Применение

Функцию RPAD можно использовать в следующих версиях Oracle / PLSQL:

  • Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i, Oracle 8i
