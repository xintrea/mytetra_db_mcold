Предварительные условия выполнения

Для выполнения настройки необходимо наличие установленного на АРМ-е клиента ald

Настройка сетевого интерфейса

vim /etc/network/interfaces

-- на примере статического адреса

auto eth0

iface eth0 inet static

      address 10.0.5.210

      netmask 24

Настройка хостов

vim /etc/hosts

10.0.5.101     ald.seacloud.local          ald                            -- сервер ald

10.0.5.210     arm-1.seacloud.local     ald-1                         -- АРМ                   

Добавление клиента в домен

ald-client join <системное имя сервера ald>

--------------------

ald-client join ald

При входе администратором на сервер ald происходит автоматическая генерация конфигурационных файлов списка добавленных в домен клиентов

Потенциальные проблемы

Разница во времени между клиентом и сервером не должна превышать 5 минут (требование Kerberos)
