Класс Floating включает в себя только числа с плавающей точкой,

то есть типы Float и Double.

Функции, которые принимают и возвращают значения, являющиеся

экземплярами класса Floating,

требуют,

чтобы эти значения

могли

быть представлены в виде числа с плавающей точкой для выполнения

осмысленных вычислений. Некоторые примеры: функции

sin,

cos

и sqrt.
