Задача «Утренняя пробежка»


Условие

В первый день спортсмен пробежал x километров, а затем он каждый день увеличивал пробег на 10% от предыдущего значения. По данному числу yопределите номер дня, на который пробег спортсмена составит не менее yкилометров.

Программа получает на вход действительные числа x и y и должна вывести одно натуральное число.


x, y, n = int(input()), int(input()), 1

while x < y: x, n = x * 1.1, n + 1

print(n)
