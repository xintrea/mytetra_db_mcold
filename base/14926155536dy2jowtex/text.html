## Работа с таблицами

Создать таблицу с первичным ключом в виде 2-ух столбцов


```sql

create table pkey(x text, y text, primary key(x,y));

```


Создать таблицу foods с внешним ключом type_id на food_types с жестким ограничением без удаления

```sql

CREATE TABLE food_types(

id integer primary key,

name text );

CREATE TABLE foods(

id integer primary key,

type_id integer,

name text );

```
