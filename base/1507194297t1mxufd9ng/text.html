HotKeys Gogland:


Ctrl + N поиск по классам


Sh + Ctrl + o поиск по файлам


Sh + Ctrl + Alt + N поиск по тексту


Ctrl + B переход к определению


Alt + 7 открыть структуру проекта


Alt + F7 использование элемента в проекте


Sh + Ctrl + I показать определение в контекстном меню


Sh + Ctrl + Space список автодополнения


Ctrl + P тип параметров функции


Sh + Ctrl + P выделение кусков кода


Alt + Enter выбор действий с некорректными записями


Sh + Ctrl + A выбор меню


Sh + F6 переименовать


Ctrl + W выделение раздела где используется элемент
