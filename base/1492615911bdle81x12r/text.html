## Commands Linux


```

df свободное пространство на дисках

free свободное пространство в оперативной памяти

pwd текущий рабочий каталог



```


### ls

```

ls ~ /usr /etc вывод содержимого нескольких каталогов

ls -l получение общей информации по файлам

ls -s размер файлов

ls -t время последнего использования

```


### file

> Отображает описание файла

```sh

file jMemorize-1.3.0.jar

jMemorize-1.3.0.jar: Java archive data (JAR)

```

### less

> Чтение содержания файла


### type

type — получение типа команды


```sh

[me@linuxbox ~]$ type type

type встроена в оболочку

[me@linuxbox ~]$ type ls

ls является алиасом для `ls --color=tty'

[me@linuxbox ~]$ type cp

cp хэширована (/bin/cp)

```


### help

```sh

mkdir --help

```


### tee

tee — чтение со стандартного ввода и запись

в стандартный вывод и в файлы

> Ниже, продолжая один из предыдущих примеров, мы сохраним полный список файлов в каталогах в файле ls.txt, перед тем как он будет отфильтрован командой grep :

```sh

[me@linuxbox ~]$ ls /usr/bin | tee ls.txt | grep zip

bunzip2

bzip2

gunzip

gzip

unzip

zip

zipcloak

zipgrep

zipinfo

zipnote

zipsplit

```
