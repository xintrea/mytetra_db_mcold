## Установка пакетов


Установка deb-пакетов

```sh

## Debian

dpkg --install файл_пакета


## Red Hat

rpm -i файл_пакета

```



```sh

sudo dpkg -i sublime-text_build-3047_i386.deb

```


## Список установленных программ

```sh

## Debian

dpkg –list

## Red Hat

rpm –qa

```


## Определение, установлен ли пакет


```sh

##Debian

dpkg --status имя_пакета


## Red Hat

rpm –q имя_пакета

```


```sh

dpkg --status emacs

```


## Вывод информации об установленном пакете


```sh

## Debian

apt-cache show имя_пакета


## Red Hat

yum info имя_пакета

```


```sh

apt-cache show emacs

```



## Поиск пакета по установленному файлу

```sh

## Debian

dpkg --search имя_файла


## Red Hat

rpm –qf имя_файла

```
