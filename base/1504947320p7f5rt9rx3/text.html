"""
Global & local variables
При использовании переменных внутри блоков(например, функций), относящихся к глобальному пространству имен
необходимо объявлять их как глобальные:
"""
# global x


x = 6

def example_func():
    # works
    print(x)
    print(x+5)

    global x                # без данной инструкции изменение глобальной переменной невозможно
    # but then what happens when we go to modify:
    x+=6
    print(x)

example_func()
