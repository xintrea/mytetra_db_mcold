Creating Strings

The most direct way to create a string is to write âˆ’

var greeting = "Hello world!"

Whenever it encounters a string literal in your code, the compiler creates a string object with its value in this case, "Hello world!'.

A string literal holds a valid UTF-8 sequences called runes. A String holds arbitrary bytes.

package main

import "fmt"

func main() {
   var greeting =  "Hello world!"
   
   fmt.Printf("normal string: ")
   fmt.Printf("%s", greeting)
   fmt.Printf("\n")
   fmt.Printf("hex bytes: ")
   
   for i := 0; i < len(greeting); i++ {
       fmt.Printf("%x ", greeting[i])
   }
   fmt.Printf("\n")
   
   const sampleText = "\xbd\xb2\x3d\xbc\x20\xe2\x8c\x98" 
   /*q flag escapes unprintable characters, with + flag it escapses non-ascii 
   characters as well to make output unambigous  
   */
   fmt.Printf("quoted string: ")
   fmt.Printf("%+q", sampleText)
   fmt.Printf("\n")  
}

This would produce the following result âˆ’

normal string: Hello world!
hex bytes: 48 65 6c 6c 6f 20 77 6f 72 6c 64 21 
quoted string: "\xbd\xb2=\xbc \u2318"

NoteÂ âˆ’ The string literal is immutable, so that once it is created a string literal cannot be changed.

String Length

len(str) method returns the number of bytes contained in the string literal.

package main

import "fmt"

func main() {
   var greeting =  "Hello world!"
   
   fmt.Printf("String Length is: ")
   fmt.Println(len(greeting))  
}

This would produce the following result âˆ’

String Length is : 12
