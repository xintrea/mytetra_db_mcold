The Bind function can also take some additional optional parameters to set more fine-grain

control on when the callback should be executed, as follows:


Bind(event, handler, source=None, id=-1, id2=-1)


The arguments to this function are described as follows:

f event: This is the event to bind to.

f handler: This is the event handler callback function to bind.

f source: This can be used to specify the window object that is the source of the

event. If specified, then only when the source object generates the event will the

handler be executed. By default, any event of the type that gets to the control will

cause the handler to execute.
