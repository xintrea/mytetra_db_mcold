IniRead, OutputVar, Filename, Section, Key [, Default]

Default - Значение, которое будет помещено в OutputVar, если запрошенный параметр не найден. Если не указано, будет использовано слово ERROR. Чтобы задать пустую строку, используйте значение %A_Space%.

Стандартный INI-файл выглядит так:

[ИмяРаздела]

Параметр=Значение


Пример test.ini:


[testSection]

m_adress = 5

angel = Angela Merkel

[hellSection]

devil = Trump



Скрипт:


F10::

IniRead, OutputVar, test.ini, testSection, m_adress

MsgBox, Значение параметра - %OutputVar%

return


F3::

IniRead, OutputVar, test.ini, testSection, angel

MsgBox, Значение параметра - %OutputVar%

return


F4::

IniRead, OutputVar, test.ini, hellSection, devil

MsgBox, Значение параметра - %OutputVar%

return
