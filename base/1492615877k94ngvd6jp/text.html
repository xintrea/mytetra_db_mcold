## Установка стандартных программ на Ubuntu


Установка SQLite 3 на Ubuntu Linux


1. Устанавливаем пакеты sqlite3 и libsqlite3-dev:


```sh

sudo apt-get install sqlite3 libsqlite3-dev

```


2. Устанавливаем gem sqlite3-ruby:

```sh

sudo apt-get install sqlite3-ruby

```

Atom

```sh

$ sudo apt-add-repository ppa:webupd8team/atom

$ sudo apt install atom

```


OpenShot - прога для видео

```sh

$ sudo add-apt-repository ppa:openshot.developers/ppa

$ sudo apt update

$ sudo apt install openshot-qt

```

Teamviewer

Double Commander

```sh

sudo add-apt-repository ppa:alexx2000/doublecmd


sudo apt-get update


Для установки QT приложения Double Commander выполните следующую команду в терминале:

sudo apt-get install doublecmd-qt


Для установки GTK приложения Double Commander выполните следующую команду в терминале:

sudo apt-get install doublecmd-gtk


из терминала можно вызвать по команде:


doublecmd

```
