CEIL ФУНКЦИЯ

Узнайте, как использовать Oracle / PLSQL функцию CEIL с синтаксисом и примерами.

Описание

Функция Oracle / PLSQL CEIL возвращает наименьшее целое число, которое больше или равно number.

Синтаксис

Синтаксис функции Oracle / PLSQL CEIL:

CEIL( number )

Параметры или аргументы

number значение используется для поиска наименьшего целого числа.

Применение

Функцию CEIL можно использовать в следующих версиях Oracle / PLSQL:

  • Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i, Oracle 8i
